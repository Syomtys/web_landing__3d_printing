# web_landing__3d_printing

# Project Description

This project is a one-page website that provides 3D printer printing services. Here you can order the printing of any 3D object, which will be carried out by professional specialists in the shortest possible time. The site provides a list of all available models, as well as the possibility of ordering an individual project.

# Instructions for deploying the project

1. Download the repository to your local computer.

2. Open the file "index.html " in any browser.
